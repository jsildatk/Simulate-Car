# Simulate-Car
JavaFX application to simulate the car.
## What can it do?
- ↑ - acceleration
- ↓ - braking
- [Space] - clutch
- [1, 2, 3, 4, 5, 6, N] - available gears
- Clutch + gear number - change gear
## Others
- Enigne braking
- Start/stop engine
- Rectangles showing your inputs
- Digital gauges to show:
  - active gear
  - engine speed (rpm)
  - actual speed (km/h)
